Windate Command Documentation

Detailed reference for supported commands in the Windate project.

/sel [client_id]

Selects a specific client by ID. All following commands apply only to the selected client.

/rename [old_id];/:[new_id]

Renames the current client if its ID matches old_idUpdates and reloads the ID to new_id

/delays [seconds]

Delays execution of the next command by the specified time in seconds.

/open "[path_or_url]"

Opens the given file path or URL using the operating system's shell.

/say "[text]"

Speaks the specified text using the system's TTS engine.

/clipset "[text]"

Copies the given text to the system clipboard.

/clipsetall "text1;/ :text2;/ :text3"

Splits text by ;/: and copies each part to the clipboard in sequence.

/beep

Triggers a system beep sound.

/show_virkey

Displays the virtual keyboard with the placeholder Hello

/hide_virkey

Hides the virtual keyboard if it is currently shown.

/alert "message;/:title"

Displays a popup alert. If only message is provided, shows it without a title. If title is included, it appears in the alert's title bar.
