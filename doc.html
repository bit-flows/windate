Windate Command Documentation

List of supported commands used by the Windate engine.

/sel [client_id]

Selects a specific client by ID. Subsequent commands apply only to the selected client.

/rename [old_id];/:[new_id]

Renames the current client if its ID matches old_id. Updates and reloads the ID.

/delays [seconds]

Delays execution for the specified time (in seconds).

/open "[path]"

Opens a file or URL using the OS shell.

/say "[text]"

Uses Text-to-Speech (TTS) to speak the provided message aloud.

/clipset "[text]"

Sets the clipboard to the given text.

/clipsetall "text1;/: text2;/: text3"

Sets the clipboard to each entry in sequence, separated by ;/:.

/beep

Plays a system beep.

/show_virkey

Displays the virtual keyboard.

/hide_virkey

Hides the virtual keyboard.

/alert "message;/: title"

Shows an alert popup. The first value is the message; the second (optional) is the title.
